# Portfolio case study drafts

Full Aurora and Rosie case study pages are saved here while the live routes show an "in progress" placeholder.

## Restore a case study

1. Copy the draft back over the live page:
   - `Aurora.full.js` → `../Aurora.js`
   - `Rosie.full.js` → `../Rosie.js`
2. Styles are unchanged in `../Aurora.css` and `../Rosie.css`.

The placeholder component lives at `../PortfolioInProgress.js`.
