# Portfolio case study drafts

Full Aurora, Rosie, and Starmap case study pages are saved here while the live routes show an "in progress" placeholder.

## Restore a case study

1. Copy the draft back over the live page:
   - `Aurora.full.js` → `../Aurora1.js` (or restore dedicated `Aurora.js` if you add one)
   - `Rosie.full.js` → `../Rosie.js`
   - `Starmap.full.js` → `../Starmap.js`
2. Styles are unchanged in `../Aurora.css`, `../Rosie.css`, and `../Starmap.css`.

The placeholder component lives at `../PortfolioInProgress.js`.
